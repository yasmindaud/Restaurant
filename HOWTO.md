# Restaurant Management System
##  Project Description
This project is a Restaurant Management System developed using Java and Object-Oriented Programming Principles.
The system manages restaurant operations such as customers, employees, menu items, and orders.
## Dynamic Input 
The program allows users to enter information dynamically using the Scanner class.
Users can enter :
- Customer name
- Contact information
- Food item name
- Food price

## Exception Handling
The system uses exception handling to prevent program crashing.
The following exception handling techniques are used:
- try block : contains code that may cause an error.
- catch block: handles errors such as invalid user input.
- finally block : ensures resources like Scanner are probably closed.

Example:
If a user enters invalid data (like letters instead of numbers for price), the system catches an error and displays a message instead of crashing.

## How to Run the Program
Compile the project:
javac *. java
Run the program:
java Main
